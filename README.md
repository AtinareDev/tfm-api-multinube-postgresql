# TFM - API Multinube con PostgreSQL sobre AWS y Azure

Este proyecto corresponde al Trabajo de Fin de Máster en Big Data, Inteligencia Artificial e Ingeniería de Datos.

## Objetivo

Diseñar e implementar una API multinube con FastAPI y PostgreSQL, simulando inicialmente dos entornos cloud en local mediante Docker:

- AWS simulada: `postgres_aws`
- Azure simulada: `postgres_azure`

La aplicación permitirá gestionar clientes, productos y pedidos, así como cambiar la cloud activa desde una aplicación cliente desarrollada con Streamlit.

## Stack tecnológico

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker
- Streamlit
- Git/GitHub
